
  
  # E-Commerce Back End

  ![license](https://img.shields.io/badge/License-Apache-brightgreen.svg)

  ## Description

   This the backend for an e-commerce website. It allows you to GET, POST, PUT, DELETE routes. The 3 routes are Categories, Products and Tags.
